Start by entering username or ID into the search field on the right.

Username search is case-sensitive (it's Chinitsu, not chinitsu), and only works if the entire username is not a number for obvious reason. In those case, please use user id instead. You can easily find it in the profile page.

Example: my profile page is https://online-go.com/user/view/197819/Chinitsu, meaning my user ID is 197819.
